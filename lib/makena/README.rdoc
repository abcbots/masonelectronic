== README

This gem is still evolving - unexpected changes should happen frequently.

Last update: Jun 30 2014

MISSION:

Flawlessly handle standard CREATE, READ, UPDATE, DESTROY, ADD, REMOVE, COPY actions;
Generate the standard has_many-through, has_many-through-referrential, and User-has_many;
Produce prototypes in minutes, rather than years.

* Ruby version: 2.1.1

* System dependencies: See Gemfile

* How to run the test suite: $ cucumber
